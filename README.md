# Quran & Prayer Website 

A religious website that provides Quran verses, Hadiths, prayer times, and Islamic educational content in a simple and clean interface.

---

##  Project Overview
This project aims to offer an all-in-one Islamic platform that helps users access Quran verses, Hadiths, prayer times, and religious Q&A easily.  
The website also includes curated Islamic YouTube lectures to enhance learning and reflection.

---

##  Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- External APIs (Quran & Hadith data)

---

##  Features
- Display Quran verses dynamically using API
- Fetch and show Hadiths from an external API
- Prayer times section
- Islamic Q&A section (common religious questions and answers)
- Embedded Islamic YouTube lecture
- Clean and simple UI
- Fully responsive layout

---

##  Concepts Used
- API Fetching using JavaScript
- DOM Manipulation
- Asynchronous JavaScript (Promises)
- Dynamic content rendering
- Clean folder structure
- Responsive design principles
